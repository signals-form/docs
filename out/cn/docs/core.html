欢迎了解 Signals Form

Signals Form 采用声明式 API 与强大的字段联动无缝集成，为开发者提供灵活的验证机制、卓越的性能表现以及对各种前端框架的全面支持。

Form Core

表单的核心引擎，提供状态管理、验证和联动能力，是整个表单系统的基础。

Form Devtools

强大的开发调试工具，帮助开发者实时监控表单状态变化，快速定位和解决问题。

Form Graph

直观的图形化工具，将表单结构和关系可视化，使复杂表单设计变得简单。

Form Relation

专业的关系处理系统，轻松处理字段间的复杂依赖和联动关系。

想探索更多功能？

查看我们详细的 核心概念文档 了解更多高级特性。

核心特性

  • ✨ 响应式状态管理：基于 Signals 的高效状态管理，自动追踪依赖关系
  • 🛡️ 强大的表单验证：灵活的校验器，支持 Zod、Yup 等主流验证库
  • 🔄 智能字段联动：轻松定义字段间的依赖关系，实现复杂的表单联动逻辑
  • 🧩 组件化设计：与 React、Vue、Svelte 等框架无缝集成
  • 🚀 极致性能：精确更新策略，避免不必要的重渲染
  • 🔗 强大的字段关联：支持复杂的字段间关系和数据流
